# User Authentication System

A simple authentication system with signup and login functionality built using Node.js, Express, MongoDB Atlas, and Mongoose.

## Features
- User signup with password hashing (bcryptjs)
- User login validation
- MongoDB Atlas integration
- Clean frontend interface

## Tech Stack
- Node.js
- Express
- MongoDB Atlas
- Mongoose

## Setup
1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with MongoDB URI and PORT
4. Run `node server.js`
5. Open `http://localhost:3000
